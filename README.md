# FastAPI : Internal Devops Utilities API
## Aim
API for getting Internal utilities using FastAPI

- AWS Service APIs
- System Metrics

## Usage
```python
$ git clone https://github.com/nithu-only/FastAPI-Python.git
$ cd FastAPI-Python
```
## To Setup Python virtual-environment
```python
$ python -m venv .env
# for windows
$ ./env/Scripts/activate.bat
# for ubuntu or mac
$ source ./env/bin/activate
```
## Install Requirements
```python
$ pip install -r requirements.txt
```
## Run App
```python
$ python main.py
```
